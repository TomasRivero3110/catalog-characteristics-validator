# Catalog Characteristics Validator

Python-based validation tool for telecom catalog characteristics and equipment configuration.

This project executes SQL queries against an Oracle database, validates equipment and catalog characteristic data, and generates Excel reports to support audit and maintenance processes.

## Features

- Execute SQL queries against Oracle databases
- Validate equipment and catalog characteristic configurations
- Compare online, partner and internal catalog data
- Export automated Excel validation reports

## Tech Stack

- Python
- SQL
- Oracle Database
- Pandas
- OpenPyXL

## Project Structure

- `src/`: Python scripts for execution and processing
- `src/`: Python scripts containing the SQL queries and validation logic
- `examples/`: Sample output reports

## Use Case

Designed for telecom catalog validation workflows where product characteristics, equipment mappings and loyalty-related data must be reviewed consistently.
